How to permanently add seasons
  1. Download the Fantrax final standings CSV for the new season
  2. Place it in the League Summary folder (same folder as the other CSVs)
  3. Add a line to YEAR_FILES in process_data.py
  4. Run python3 process_data.py from the baseball-app folder
  5. Commit and push all changes to GitHub — the site updates automatically
